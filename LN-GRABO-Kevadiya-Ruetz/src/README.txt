Anleitung zum Ausführen der .jar-Datei über die Windows-Kommandozeile:

Es funktioniert nicht anders, wie sie es uns angegeben haben:

java ‐‐module‐path "C:\path-to-javafx-directory\lib" ‐‐add‐modules javafx.controls,javafx.fxml -jar [Pfad, an dem sie die .jar-Datei gespeichert haben]

Es wird zum Ausführen openjdk version 17.0.1 und javafx sdk version 17.0.1 benötigt, ebenso wie sie es angegeben haben.
